# 📊 Sales Analysis Dashboard (Power BI)

## 📌 Overview

This project is a Power BI dashboard for analyzing sales performance.

## 🎯 Objective

To visualize and analyze sales data to gain business insights.

## 📊 Features

* Sales trends over time
* Top performing products
* Region-wise sales analysis
* Profit insights

## 🛠️ Tools Used
